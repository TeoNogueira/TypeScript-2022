Antes de tudo mude a configuração do arquivo tsconfig.json, dentro de "compilerOptions" : {

    "target": "es5" para "es6";

}

se você observar o arquivo gerado em js ele utiliza uma modelagem antiga do javascript ex: var no lugar de let.


 (Comando usado ex: tsc compiler.ts)
